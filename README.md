# rocketshoes
App para e-commerce utilizando Redux

# Getting Started

Installation

Clone the repo:

  $ git clone https://github.com/vinic-asouza/rocketshoes.git
  
Install Yarn packages:

  $ yarn

Run JSON-Server:

  $ json-server server.json -p 3333
  
Run Web Application (ReactJS):

  $ yarn start
